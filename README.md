# ADLabPSTU

Лабораторная работа по андроид-разработке

Приложение для владельцев кошек

Авторы/разработчики: Волкова Ксения, Филатова Анна, Жеребцов Матвей

1. Базовый функционал:<br>
- Экран состояния питомца <br>
(ввод и отображение описания животного, а именно: имени, породы, даты рождения и веса)<br><br>
- Дневник здоровья/питания <br>
(календарь с возможностью делать записи о внешнем состоянии животного, о его приёмах пищи и воды)<br>
- Подсчёт несложных показателей <br>
(индекс массы тела, текущий возраст)<br>
2. Варианты продвинутого функционала:<br>
- Работа с картой <br>
(поиск и отображение через неё ближайших зоомагазинов, ветаптек, приютов)<br>
- Социальное взаимодействие <br>
(публичные профили пользователей с возможностью постинга фото/видео, возможность оставлять реакции в виде оценок (лайков) и комментариев)<br>
- Экспертная система и/или статистика (на основании данных система сможет выдавать пользователю советы <br>
(например, если рацион животного превышает норму, то система рекомендует его снизить)<br>
 Для выдачи ответов может потребоваться сбор статистики. Собранные показатели могут быть выведены для пользователя)<br>
